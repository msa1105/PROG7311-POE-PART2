# Database Schema - GLMS

This SQL script contains the complete database schema generated by Entity Framework Core migrations for the Global Lecturer Management System (GLMS).

## Database Structure

### Tables Created:
1. **Clients** - Educational institutions and organizations
2. **Contracts** - Agreements between clients and the system
3. **ServiceRequests** - Individual service requests tied to contracts

### Relationships:
- One Client can have many Contracts (1:N)
- One Contract can have many ServiceRequests (1:N)

### Constraints:
- Foreign key from Contracts to Clients (ON DELETE RESTRICT)
- Foreign key from ServiceRequests to Contracts (ON DELETE CASCADE)

## Migration Information

**Generated On**: 15 January 2025
**EF Core Version**: 9.0.5
**Target Framework**: .NET 9.0

## Usage Instructions

### Option 1: Apply using EF Core CLI
```bash
dotnet ef database update
```

### Option 2: Execute SQL Script Manually
Run the generated `Database_Schema.sql` file against your SQL Server instance using:
- SQL Server Management Studio (SSMS)
- Azure Data Studio
- sqlcmd command-line tool

```bash
sqlcmd -S (localdb)\mssqllocaldb -i Database_Schema.sql
```

### Option 3: Use Visual Studio Package Manager Console
```powershell
Update-Database
```

## Notes

- This script is **idempotent** - it can be run multiple times safely
- All date fields use SQL Server `datetime2` type
- Decimal fields use `decimal(18,2)` precision
- String length constraints are enforced via `nvarchar` types
- The script includes checks to avoid recreating existing objects
